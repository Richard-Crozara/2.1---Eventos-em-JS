O atributo ondblclick é utilizado em HTML para definir uma ação que deve ocorrer quando um elemento é clicado duas vezes consecutivas com o botão esquerdo do mouse. Esta ação pode ser um script JavaScript que manipula o elemento, como exibir uma mensagem na tela ou mudar o conteúdo do elemento, entre outras possibilidades. O ondblclick é um evento de clique que só é acionado quando o usuário clica duas vezes consecutivas no elemento.
