O evento "onchange" é um evento em JavaScript que é acionado quando o valor de um elemento HTML, como um input ou select, é alterado pelo usuário. Ele é frequentemente utilizado para atualizar uma página web com base nas escolhas do usuário ou para executar uma ação quando um determinado elemento é alterado, Por exemplo, você pode usar o evento onchange para atualizar uma tabela de preços quando um usuário seleciona um produto em um menu suspenso ou para validar se um valor digitado em um campo de formulário é válido.
